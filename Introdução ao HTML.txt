Introdução ao HTML

Geral:
-Tecnologia front-end para criar websites estáticos
-Usado para criar conteúdo textual
-Linguagem de marcação

arquivo: exemplo.html

Sintaxe básica:
-HTML ignora "espaços" duplos, triplos...
-HTML não entende o "enter" para mudar de linha, tem que usar uma 'tag' (<p>exemplo</p>)
-A maioria das tags tem que ter um comando para abrir e outro para fechar
-Podemos adicionar atributos nas tags
